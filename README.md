# Paradise Nursery Shopping Cart

A React + Redux Toolkit shopping cart for an online plant shop. The app includes a landing page, product listing by category, and a cart page with full quantity controls.

## Features
- Landing page with a call-to-action button
- Navigation bar with cart quantity badge
- Product listing with Aromatic and Medicinal sections
- Add to cart, increase/decrease quantity, and delete items
- Live total cost updates

## Tech Stack
- React
- React Router
- Redux Toolkit
- Vite

## Getting Started

```bash
npm install
npm run dev
```
